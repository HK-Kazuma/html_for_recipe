ターミナルで
フォルダを操作してみる

エクスプローラーのクリック操作を、文字のコマンドに置き換えるだけ。

1
エクスプローラーとターミナル、何が違う？
「同じことをしている」という前提でまず理解する

やっていることは実は同じ。「マウスでやるか」「文字で命令するか」の違いだけ。

🖱️
エクスプローラー（Finder）
フォルダのアイコンをダブルクリックして開き、目で見て操作する方法。
✅ 見たまま操作できる、直感的
✅ 初めてでも迷わない
⚠️ 同じ作業を10回繰り返すのは面倒
⚠️ 人に「操作の手順」を正確に伝えにくい
≒
⌨️
ターミナル（コマンド操作）
「これをやって」という命令を文字で1行書いて実行する方法。
✅ 同じ作業を一瞬で繰り返せる（コピペでOK）
✅ 手順を「文字」として正確に共有できる
✅ プログラムの実行・GitHub操作は基本ターミナル
⚠️ 最初は「今どこにいるか」が見えにくい

なぜ今ターミナルを覚える必要があるのか：Next.jsの起動（npm run dev）もFastAPIの起動もGitHubへのpush/pullも、全部ターミナルから行う。エクスプローラーではこの先の開発作業がほぼ進められないので、「フォルダ移動」だけでも先にターミナルでできるようにしておく。

2
「今どこにいるか」という感覚をつかむ
フォルダは入れ子になった箱。今その箱の中のどこにいるかが常に決まっている

📁 エクスプローラーでの見え方

📂 want-cooking/
📂 frontend/
📄 package.json
📂 backend/
📄 main.py

⌨️ ターミナルでの見え方（今いる場所が文字で出る）

~/want-cooking $
# ↑ 今 want-cooking フォルダの中にいる
~/want-cooking/backend $
# ↑ backendフォルダの中に移動した後
💡

迷子になったら：とにかくpwd（今いる場所を表示）とls（中身を表示）の2つを打てば、エクスプローラーで画面を見るのと同じ状況を文字で再現できる。

3
覚える6つのコマンド
エクスプローラーの操作との対応で覚える
📍 pwd
今どこにいるか表示する エクスプローラーの「今開いているフォルダの場所」を確認するのと同じ
$ pwd
→ /Users/kazuma/want-cooking
📋 ls
中身の一覧を表示する エクスプローラーでフォルダを開いたときに見える「中のファイル・フォルダ一覧」
$ ls
frontend backend README.md
➡️ cd 名前
指定したフォルダの中に移動する エクスプローラーで「フォルダをダブルクリックして開く」動作と同じ
$ cd backend
# backendフォルダの中に移動した
⬅️ cd ..
1つ上のフォルダに戻る エクスプローラーの「戻る」ボタン・「上の階層へ」ボタンと同じ
$ cd ..
# 1つ上の階層に戻った
🗂️ mkdir 名前
新しいフォルダを作る エクスプローラーの「右クリック→新規フォルダ」と同じ
$ mkdir images
# imagesという名前の空フォルダができた
🗑️ rm -r 名前
フォルダを中身ごと削除する エクスプローラーの「削除」に近いが、ゴミ箱を経由せず即座に消える
$ rm -r old_test
# old_testフォルダが中身ごと完全に消えた
⚠️

rm -rは取り消せない：エクスプローラーの削除は「ゴミ箱」から復元できるが、ターミナルのrm -rは復元不可。実行前にlsで「今どこにいて」「何を消そうとしているか」を必ず確認する癖をつける。

4
実際に組み合わせて使ってみる
「backendフォルダの中に新しいフォルダを作る」を一連の流れで
1
pwd
今どこにいるか確認する
→ /Users/kazuma/want-cooking にいるとわかった
2
ls
中に何があるか確認する
→ frontend / backend フォルダがあるとわかった
3
cd backend
backendフォルダの中に移動する
4
mkdir images
imagesという新しいフォルダを作る
5
cd ..
1つ上に戻って、また別の作業に移る
5
まとめ・チートシート
迷ったらここに戻る
見るだけの操作 → pwd / ls 移動する操作 → cd / cd .. 作る・消す操作 → mkdir / rm -r
コマンドやることエクスプローラーで言うと
pwd今いる場所を表示するアドレスバーを見る
ls今いる場所の中身一覧を表示するフォルダを開いて中身を見る
cd 名前指定したフォルダに移動するフォルダをダブルクリックする
cd ..1つ上の階層に戻る「戻る」ボタンを押す
mkdir 名前新しいフォルダを作る右クリック→新規フォルダ
rm -r 名前フォルダを中身ごと削除する削除（ただしゴミ箱には入らない）
1

ターミナルを開いてpwdとlsだけ打ってみる

今の場所と中身を確認する、これだけでも第一歩

2

デスクトップにテスト用フォルダを1つmkdirで作ってみる

エクスプローラー側を開いて、本当にフォルダができているか目で確認する

3

作ったテスト用フォルダをrm -rで消してみる

練習用の空フォルダで、消える感覚に慣れておく
